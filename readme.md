## about
simple restful dotnet core 5.0 CRUD with in-memory database

---

## how-to
* run dotnet build
* run dotnet run

## routes
```[GET] /api/users```

```[GET] /api/user/{id}```

```[POST] /api/users```

```[POST] /api/users/login```


## example body request
```
{
    "email": "hello@world.com",
    "password": "hello_world"
}
```
